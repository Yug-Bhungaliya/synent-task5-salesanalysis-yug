# Sales Data Analysis

## Objective
Analyze Superstore sales data to identify:
- Monthly revenue trends
- Top-selling products
- Profit analysis

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Project Workflow
1. Data Cleaning
2. Exploratory Data Analysis
3. Visualization
4. Business Insights

## Key Insights
- Sales fluctuate monthly
- Technology products have highest sales
- Some categories have low profit margins

## Conclusion
This project helps understand business performance and supports data-driven decision making.
