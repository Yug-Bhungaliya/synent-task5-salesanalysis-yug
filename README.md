# Sales Data Analysis Dashboard

## 📌 Project Overview
This project is developed as part of the Synent Technologies Data Science Internship Program.

The objective of this project is to analyze Superstore sales data and generate meaningful business insights using Python, data visualization, and dashboard creation.

The project includes:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Business Insights
- Interactive Dashboard using Streamlit

---

# 📂 Dataset
Dataset Used:
- Superstore Sales Dataset

Dataset Source:
https://www.kaggle.com/datasets/vivek468/superstore-dataset-final

---

# 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Streamlit
- Jupyter Notebook

---

# 📊 Project Workflow

## 1. Data Cleaning
- Checked missing values
- Removed duplicate rows
- Converted date columns into proper format

## 2. Exploratory Data Analysis
Performed analysis on:
- Monthly Revenue Trends
- Top Selling Products
- Profit Analysis
- Region-wise Sales
- Customer-wise Analysis

## 3. Data Visualization
Created different charts and graphs for better understanding of business performance.

## 4. Dashboard Creation
Developed an interactive dashboard using Streamlit.

---

# 📈 Key Insights
- Sales fluctuate across different months
- Technology products generate the highest sales
- Some categories have high sales but low profit margins
- Region-wise analysis helps identify high-performing areas
